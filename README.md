Steps

1. clone this repo
2. inside your project folder, create a .env file
3. put claude api key in it
4. and run the app.py
5. open chrome, go to manage extensions > on developer mode > click on load unpack > select your project folder and it will load the extension
6. go to linkedin.com on chrome and will see the buttons in the post comment section
7. click on AI comment button, wait for a few seconds and you will see the comment
8. click on send button to post the comment and save the comment as well in json file for future use
